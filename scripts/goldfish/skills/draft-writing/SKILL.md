# Draft Writing Skill

## Purpose

Convert useful intelligence into safe, reviewable Obsidian draft notes.

## When To Use

Use this skill when creating permanent note drafts, prompt drafts, project inspiration drafts, or structured knowledge-deposition reports.

## Safety Rules

1. Drafts are candidate material, not final truth.
2. Always preserve source links.
3. Do not invent claims, quotes, metrics, or opinions.
4. Mark uncertain items as needing verification.
5. Keep the user's own judgment section empty or clearly labeled as a prompt for review.

## Draft Structure

- Frontmatter
- Title
- Source
- Core information
- Why it matters
- Value for me
- Reusable insight
- Next actions
- Raw summary

## Quality Bar

A draft is worth writing only when it can become at least one of:

- A permanent note
- A business idea
- A reusable prompt
- A project idea
- A technical decision note
- A future research question
