# Trend Analysis Skill

## Purpose

Turn daily AI intelligence into clear trend judgments.

## When To Use

Use this skill for questions about AI trend direction, repeated signals, weekly review, or what deserves deeper study.

## Analysis Frame

1. Identify repeated signals across sources.
2. Separate facts from interpretation.
3. Group signals into themes.
4. Compare momentum, novelty, and relevance to the user's focus.
5. Explain uncertainty and missing evidence.
6. Suggest what to watch next.

## Priority Themes

- Agent
- RAG
- AI Coding
- Codex-like tools
- Claude Code-like tools
- MCP
- Knowledge base systems
- AI application development
- Technical monetization

## Output Shape

For each trend:

- Trend name
- Evidence
- Why it matters
- Who is affected
- Opportunity
- Risk
- Suggested next action
- Suggested note location
