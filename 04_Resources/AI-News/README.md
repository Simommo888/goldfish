# AI-News

这里存放 AI 情报 Agent 生成和维护的公开 AI 信息。

## 目录说明

1. `Daily`：每日 AI 情报日报。
2. `Weekly`：每周 AI 趋势总结。
3. `People-Watch`：AI 关键人物公开专业动态。
4. `Raw`：原始抓取缓存 JSON。
5. `Sources`：信息源维护说明。
6. `Reports`：扩展分析报告。

## 如何把新闻变成永久笔记

当一条新闻包含长期有效的判断、方法或框架时，把它改写成自己的话，放入 `05_Permanent-Notes`，并链接相关 MOC。

## 如何把新闻变成商业想法

当一条新闻暴露了客户需求、付费场景、企业痛点、效率提升机会时，沉淀到 `11_Business-Ideas`。

## 如何把新闻变成项目灵感

当一条开源项目、论文或产品可以改造成自己的 Demo、工具或作品集项目时，沉淀到 `02_Projects`。

## 如何把新闻变成 Prompt

当一条内容能提炼出可复用工作流、检查清单或写作/开发提示词时，沉淀到 `09_Prompts`。

## 使用原则

- 只沉淀公开、可靠、可复用的信息。
- 不沉淀八卦、私人生活、无来源爆料。
- 每天最多精处理 3 条，避免只收藏不复用。
